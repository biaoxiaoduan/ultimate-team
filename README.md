# ultimate-team

AI Agent 开发团队管理平台。

## 当前状态

当前仓库处于产品方案阶段，已完成首版产品方案文档：

- [PRODUCT_PLAN.md](./PRODUCT_PLAN.md)

该方案已收敛为单用户优先版本，适合先由单人开发和使用，后续再逐步扩展到多人协作和更复杂的 Agent 调度能力。

## 产品目标

围绕以下核心链路构建产品：

- 项目管理
- 文档管理
- 迭代与任务管理
- 测试记录与回归跟踪
- AI 任务拆解与报告生成
- Git / CI 状态关联

## 推荐开发顺序

1. 初始化前后端项目骨架
2. 落库核心数据模型
3. 实现项目、文档、任务、测试基础能力
4. 接入 AI Agent 能力
5. 接入 Git / CI 集成

## 仓库约定

- `PRODUCT_PLAN.md`：产品方案基线文档
- 后续建议目录：
  - `apps/web`：前端应用
  - `apps/api`：后端应用
  - `packages/shared`：共享类型和工具
  - `docs/`：补充设计文档

## 下一步

建议下一步直接开始初始化项目骨架，并根据 `PRODUCT_PLAN.md` 拆分 MVP 开发任务。
