# 迭代 5 测试用例

## 1. 测试范围

- 流程型页面导航
- 新建需求 / Agent / Run 弹窗
- Run 到交付物的跳转
- Artifact 列表和详情查看
- TestReport 和 BuildRecord 展示
- 原有需求、计划、Agent、Run 流程回归

## 2. 不测试范围

- 真实浏览器地址栏深链恢复
- 真正的 Git 提交抓取
- 真实 CI/CD 调度

## 3. 功能测试用例

### 3.1 页面流程

1. 进入工作台后可跳转到项目概览
2. 项目概览页可跳转到需求中心、Agent 中心、Run 中心、交付物中心
3. 需求列表可进入需求详情页
4. Run 列表可进入 Run 详情页
5. 交付物列表可进入 Artifact 详情页

### 3.2 创建弹窗

1. 需求创建弹窗可成功提交并跳转到需求详情
2. Agent 创建弹窗可成功提交并回到 Agent 列表
3. Run 创建弹窗可成功提交并跳转到 Run 详情

### 3.3 交付物闭环

1. 阶段完成后可在交付物中心看到对应 Artifact
2. 测试阶段完成后能看到 TestReport
3. 发布阶段完成后能看到 BuildRecord
4. 从 Run 详情页可跳转到交付物页

## 4. 异常流程用例

1. Artifact 不存在时详情页展示错误提示
2. 没有任何交付物时交付物页展示空状态
3. 缺少已完成阶段时，不生成对应 Artifact

## 5. 回归范围

- 需求录入与版本管理
- 计划生成与确认
- Agent CRUD
- Run 创建、启动、执行、确认、失败、重试、暂停、继续

## 6. 自动化测试覆盖目标

- 后端覆盖 Artifact、TestReport、BuildRecord 生成与查询
- 前端覆盖页面导航、创建弹窗和交付物查看链路
