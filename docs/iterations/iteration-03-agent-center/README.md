# 迭代 3：Agent 中心与角色模板

## 目标

把 Agent 作为一等对象纳入系统，支持角色模板、Agent 实例、Provider 绑定以及启停和删除管理。

## 范围

- AgentTemplate 模型
- AgentInstance 模型
- 内置 5 个角色模板
- Agent 实例创建、编辑、启停、删除
- Agent 与 Provider 绑定
- Agent 系统提示词和任务类型配置
- 前端 Agent 中心页面

## 不包含

- 多 Agent 编排
- Agent 自动执行代码
- Agent 运行日志
